RAIDidator

A simple Programm to Calculate HDD-Space usage with different RAID Levels

View project on GitHub

About

RAIDiator can be used to compare various RAID-configurations. RAIDiator can calculate:

  • The usable space for a certain RAID-configuration.
  • The not usable and thatway wasted space.
  • The ratio of usable and not usable space shown as a pie-diagram. More features are comming!
